# 🚀 Portfólio Dev - Projeto de Estudos

Este projeto foi desenvolvido **exclusivamente para fins de estudo e prática**, seguindo um layout inspirado no Figma disponibilizado pela Rocketseat. O objetivo principal foi treinar lógica de desenvolvimento web, estruturação semântica, organização de arquivos, boas práticas de CSS e resolução de problemas comuns de front-end.

---

## 📚 O que foi praticado?

- Estruturação HTML5 semântica (section, header, etc)
- CSS moderno: Flexbox, Grid, variáveis, responsividade
- Organização de pastas e caminhos relativos de imagens/assets
- Uso de SVG e customização de ícones por cor via CSS
- Fontes externas (Google Fonts: Archivo e Maven Pro)
- Estruturação modular de CSS (separação por seção)
- Reprodução fiel de layout do Figma (cores, espaçamentos, fontes)
- Debug de erros de caminho e deploy local

---

## 🛠️ Tecnologias Utilizadas

- HTML5 + CSS3 puro
- SVG para ícones
- [Figma](https://www.figma.com/) (referência do layout)
- VSCode + Live Server

💡 Aprendizados
- Importância de caminhos relativos em projetos estáticos
- Organização e clareza dos seletores CSS
- Utilizar fontes externas e responsividade para deixar o layout fiel ao design
- Prática real de debug (DevTools, Network, erros 404)
- Separar CSS por seção/página, como profissionais fazem em projetos grandes
 🚩 Observações

- **Este projeto não é para fins de portfólio profissional ou uso comercial.**  
  Foi desenvolvido apenas como exercício de aprendizado, sem backend ou dados dinâmicos.
- As imagens, ícones e layout são de uso educacional, baseados em materiais públicos da Rocketseat.
